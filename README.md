*** PlatinumRx Data Analyst Assignment

## Overview
This repository contains complete solutions for the PlatinumRx DA Assignment covering SQL, Spreadsheets, and Python.

## Repository Structure:
- SQL/ — Hotel and Clinic management SQL queries
- Spreadsheets/ — Ticket Analysis Excel file
- Python/ — Time converter and duplicate remover scripts

## SQL Section:
### Hotel Management (Section A):
- Q1: Last booked room per user
- Q2: Total billing per booking in November 2021
- Q3: Bill amounts greater than 1000 in October 2021
- Q4: Most and least ordered item per month in 2021
- Q5: Customers with second highest bill per month in 2021

### Clinic Management (Section B):
- Q1: Revenue by sales channel for year 2021
- Q2: Top 10 most valuable customers for year 2021
- Q3: Month-wise revenue, expense, profit and status for 2021
- Q4: Most profitable clinic per city for given month
- Q5: Second least profitable clinic per state for given month

## Spreadsheet Section:
- Populated ticket_created_at column using INDEX formula
- Outlet-wise count of tickets created and closed same day
- Outlet-wise count of tickets created and closed in same hour

## Python Section
- Q1: Convert minutes into human readable format
- Q2: Remove duplicate characters from string using loop

## Tools Used
- MySQL 8.0 for SQL queries
- Google Sheets for spreadsheet tasks
- Python 3 for scripting
